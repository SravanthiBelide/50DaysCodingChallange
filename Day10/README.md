# &#x20;Instagram Engagement Analysis:



###### Dataset:

• Post\_ID – Unique identifier of the post 

• Post\_Date – Date of the post 

• Post\_Type – Image, Video, Reel, Story 

• Caption\_Length – Number of words in caption 

• Hashtags\_Used – Count of hashtags 

• Likes – Number of likes received 

• Comments – Number of comments received 

• Shares – Number of shares 

• Saves – Number of saves 

• Reach – Total number of unique users reached 

• Impressions – Total number of views (including repeats) 

• Follower\_Count – Total followers at the time of posting
